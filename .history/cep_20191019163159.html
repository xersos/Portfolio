Credo-Crues
Site de gestion de crues
  • Release date: 2019
  • Tags: Flood management
  • Producer: Gauthier Vuillemin/ Thomas Tosch/ Jacob Corentin

Credo-Crues est un site de référencement de crues qui ont eu lieu en France. Celui-ci est prévu afin d'être un support pour référencer des informations tel que des articles réalisés par différents chercheurs !

Notre groupe a pour mission de fournir un site web, permettant l'ajout d'articles. Pour cela nous avons besoin d'un espace administrateur nécessaire à la bonne gestion du site. Mais aussi un espace "user" pour mettre aux personnes d'avoir accès aux différentes ressources. Le site est réalisé en Symphonie (mettant à disposition une api), pour la partie Back-end. Nous avons donc réalisé une refonte totale du site d'origine. Le back-end (php) a été remanié pour le rendre disponible en tant qu'API. Et une refonte du front-end a été réalisé pour passer sur le Framework Angular.

Technologies utilisées: Angular 7, Symphonie API + PostgreSQL + ApiPlatform

Ma Contribution Réalisation de la structure sous Angular + Implémentation de l'api côté client. Création de services backs pour l'api.
